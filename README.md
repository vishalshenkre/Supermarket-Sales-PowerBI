# Supermarket-Sales-Analysis-Dashboard

## Project Overview
This project analyzes supermarket sales data using Power BI.
The objective is to identify sales trends, branch performance,
product line performance, and customer behavior.

## Tools Used
- Power BI
- Power Query
- DAX

## Key KPIs
- Total Revenue
- Total Quantity Sold
- Average Rating
- Branch-wise Sales

## Insights
- Highest revenue branch identified
- Monthly sales trend analyzed
- Product line contribution evaluated
